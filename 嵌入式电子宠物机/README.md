# 🐱 基于 STM32 的电子宠物（Virtual Pet）

## 📖 项目简介

本项目基于 **STM32F103C8T6** 开发，使用 OLED 显示屏实现一个简单的电子宠物。

目前已经实现宠物状态管理、情绪系统、按键交互、蜂鸣器提示以及多界面显示，后续将继续加入等级成长、动画、原创点阵宠物、小游戏等功能。

---

# ✨ 已完成功能

## ✅ 硬件驱动

- OLED 显示
- 三个按键
- 蜂鸣器
- TIM2 定时器（1ms SystemTick）
- LED 驱动

---

## ✅ 宠物系统

支持以下属性：

- Food（饱食度）
- Happy（快乐值）
- Health（生命值）
- Level（等级）
- Alive（存活状态）

系统每秒自动更新一次宠物状态：

- Food 自动减少
- Happy 自动减少
- Food 或 Happy 过低时开始掉血
- Health 为 0 时宠物死亡

---

# 😊 情绪系统（Mood）

目前支持五种情绪：

| 情绪 | 表情 |
|------|------|
| Normal | (^_^) |
| Happy | (^o^) |
| Hungry | (T_T) |
| Sick | (-_-) |
| Dead | (x_x) |

情绪会根据宠物状态自动更新。

---

# 🖥 UI 界面

目前共有四个界面：

### PET 界面

显示宠物当前表情。

- 开机显示 5 秒
- 喂食后显示 2 秒
- 玩耍后显示 2 秒
- 治疗后显示 2 秒
- 宠物进入饥饿或生病状态时自动弹出

---

### 状态界面

实时显示：

- Food
- Happy
- HP
- Level

---

### Level Up

（界面已预留，等待升级系统完成。）

---

### Game Over

当 Health 为 0 时进入 Game Over 界面。

---

# 🔔 蜂鸣器

按键操作：

- 喂食
- 玩耍
- 治疗

都会播放一次提示音。

当：

- Food < 20
- Happy < 20

蜂鸣器会每秒报警一次。
