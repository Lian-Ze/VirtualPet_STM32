# 🐱 基于 STM32 的电子宠物（Virtual Pet）

## 📖 项目简介

本项目基于 **STM32F103C8T6** 开发，使用 **128×64 OLED（I2C）** 显示屏实现一个简单的电子宠物（Virtual Pet）。

项目采用模块化设计，实现了宠物成长、情绪管理、等级系统、多界面显示、按键交互、蜂鸣器提示、眨眼动画以及自动睡眠等功能，并通过 **TIM2** 提供 1ms SystemTick 作为系统时基。

---

# ✨ 已完成功能

## ✅ 硬件驱动

* OLED 显示（128×64）
* 三个按键输入
* 蜂鸣器
* TIM2 定时器（1ms SystemTick）
* LED 驱动

---

## ✅ 宠物系统

目前支持以下属性：

* Food（饱食度）
* Happy（快乐值）
* Health（生命值）
* Level（等级）
* Exp（经验值）
* Alive（存活状态）

系统每秒自动更新一次宠物状态：

* Food 自动下降
* Happy 自动下降
* Food 或 Happy 过低时开始掉血
* Health 为 0 时宠物死亡
* 睡眠状态下暂停属性下降

---

## 😊 情绪系统（Mood）

目前支持五种情绪：

| Mood   | 宠物表情      |
| ------ | --------- |
| Normal | `(=^.^=)` |
| Happy  | `(=^o^=)` |
| Hungry | `(=T.T=)` |
| Sick   | `(=-.-=)` |
| Dead   | `(=x.x=)` |

宠物情绪会根据当前状态自动更新，并在状态发生变化时自动切换到 PET 界面。

---

## ⭐ 等级成长系统

宠物可以通过以下操作获得经验值（EXP）：

* 🍖 喂食（Feed）
* 🎮 玩耍（Play）
* 💊 治疗（Heal）

经验值达到升级条件后：

* Level +1
* Food 恢复
* Happy 恢复
* Health 恢复
* 播放升级动画
* 蜂鸣器双响提示

---

## 😴 睡眠系统

为了模拟真实电子宠物，加入了自动睡眠功能。

功能特点：

* 长时间无操作自动进入睡眠
* 睡眠状态显示 `Zzz`
* 任意按键可立即唤醒
* 睡眠期间暂停 Food、Happy、Health 的自动下降

---

## 😉 眨眼动画

宠物会自动进行眨眼动画。

功能特点：

* 定时自动眨眼
* 正常状态下循环播放
* 睡眠状态停止眨眼

# 🖥 UI 界面

目前共有 **5 个界面**。

## 🐱 PET

显示宠物当前状态。

根据 Mood 自动切换 ASCII 萌猫表情：

### 😊 Normal

```text
   /\_/\
  (=^.^=)
   (___)
```

### 😄 Happy

```text
   /\_/\
  (=^o^=)
   (___)
```

### 😭 Hungry

```text
   /\_/\
  (=T.T=)
   (___)
```

### 🤒 Sick

```text
   /\_/\
  (=-.-=)
   (___)
```

### 😴 Sleep

```text
   /\_/\
  (=-.-=)
    Zzz
```

### ☠ Dead

```text
   /\_/\
  (=x.x=)
   (___)
```

自动显示规则：

* 开机显示 5 秒
* 按键操作后显示 2 秒
* 宠物进入异常状态自动弹出
* 睡眠状态保持显示直到被唤醒

---

## 📊 STATUS

实时显示：

* Food
* Happy
* HP
* Level
* Exp

---

## 🎉 LEVEL UP

升级成功后显示：

```text
LEVEL
 UP!
```

同时显示当前等级。

---

## 😺 LEVEL PET

升级动画结束后：

显示开心状态的小猫。

随后自动返回 STATUS 页面。

---

## ☠ GAME OVER

当 Health 为 0 时：

```text
GAME
OVER
```

宠物停止更新。

# 🔔 蜂鸣器

### 普通操作

* 喂食
* 玩耍
* 治疗

播放一次提示音。

### 升级

播放双响提示音。

### 报警

当：

* Food < 20
* Happy < 20

蜂鸣器每秒报警一次。
